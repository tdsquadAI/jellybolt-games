## 4. Enable Monetization (Revenue Share)

Game Jolt offers a **revenue share program** for game developers.

### Steps to Enable

1. Go to **Dashboard → Game → Manage → Pricing**
2. Set game to **Paid** or **Pay What You Want** for eligible games
3. Navigate to **Account Settings → Earnings** → connect a payment method (PayPal or bank transfer)
4. Game Jolt takes a **10% platform fee**; developers receive **90%**

### Recommended Pricing

| Game | Model | Suggested Price |
|------|-------|----------------|
| Dungeon Bolt | Paid | $2.99 |
| Hex Match | Pay What You Want | Min $0, suggested $1.00 |
| Neon Snake | Pay What You Want | Min $0, suggested $1.00 |
| Asteroid Dash | Pay What You Want | Min $0, suggested $1.00 |
| Rhythm Tap | Pay What You Want | Min $0, suggested $2.00 |

## 5. Optimization Tips

- **Screenshots:** Upload at least 3 screenshots per game (recommended: 1280×720 or 1920×1080)
- **Thumbnail:** Square image 200×200 minimum for the game card
- **Header:** 1000×350 banner for the game page header
- **Description length:** 150–500 characters for best display in search
- **Tags:** Use all 8 tag slots — include `html5` and `browser` on every game
- **Categories:** Select the most specific genre (Puzzle, Arcade, RPG, etc.)
