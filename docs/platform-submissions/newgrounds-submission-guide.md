## 2. HTML5 Upload Process

Newgrounds supports HTML5 games uploaded as a **ZIP file** or via **external URL**.

### Method A: ZIP Upload (Recommended)

1. Go to **https://www.newgrounds.com/upload/create**
2. Select **Games** → **HTML5**
3. Package the game:
   ```
   game-name/
   └── index.html   (single file — all JellyBolt games are self-contained)
   ```
   Zip the `index.html` and name it `{game-name}.zip`
4. Upload the ZIP
5. Set the **Canvas Size** to match each game (most JellyBolt games use 800×600 or full-window)

### Method B: External URL / Iframe

Newgrounds also accepts an **external URL** (iframe embed):
1. Select **External URL** during upload
2. Enter: `https://tamirdresher.github.io/jellybolt-games/games/{game-slug}/`
3. Note: external URL submissions have less discoverability than uploaded ZIPs

**Recommendation:** Use ZIP upload for the top 4 priority games; external URL for the rest.

## 4. Newgrounds Loader — Ad Revenue

Newgrounds offers a **revenue share program** through the **Newgrounds Loader** (formerly `io.newgrounds.com`).

### How It Works

- Newgrounds shows display and video ads around your game
- Revenue is split: developers receive a share of ad impressions generated by their game
- **No SDK required for HTML5 games** — ads appear on the game page frame, not in-game
- Earnings are tracked in the **Creator Funds** dashboard

### Optional: Newgrounds API (Medals + Scoreboards)

For deeper integration and higher engagement:

1. Register your game in the **NG App Hub**: https://www.newgrounds.com/account/games
2. Get your **App ID** and **Encryption Key**
3. Add the Newgrounds.io JavaScript library:
   ```html
   <script src="https://newgrounds.io/api/core/newgrounds_io.js"></script>
   ```
4. Initialize in your game:
   ```javascript
   ngio = new NGIO("YOUR_APP_ID", "YOUR_ENCRYPTION_KEY");
   ```
5. Add **Medals** (achievements) and **Scoreboards** — these increase discoverability significantly on Newgrounds

### Enable Creator Fund Payments

1. Go to: **https://www.newgrounds.com/account/wallet**
2. Connect PayPal for withdrawals
3. Minimum withdrawal: $20
